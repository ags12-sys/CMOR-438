# Wine Quality Dataset
## Description

Red and white wine variants of the Portuguese "Vinho Verde" wine used for the assessment of quality. Found in the wine+quality.zip folder.

## Data Files

-winequality-red.csv

-winequality-white.csv

## Features (Input Variables)

-Fixed acidity

-Volatile acidity

-Citric acid

-Residual sugar

-Chlorides

-Free sulfur dioxide

-Total sulfur dioxide

-Density

-pH

-Sulphates

-Alcohol

# California Housing Dataset
## Description

Based on 1990 California census data used to predict housing values. Found under the archive(1).zip folder.

## Features (Input Variables)

-Median income

-House age

-Average rooms

-Average bedrooms

-Population

-Average occupancy

-Latitude

-Longitude
